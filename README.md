# Scale codec golang

Scale Codec written by golang, reference from https://github.com/polkascan/py-scale-codec

## Features

- [x] low level codec decode
- [x] metadata decode
- [x] extrinsic decode
- [x] LogDigest decode
- [x] event decode
- [x] custom type reg
- [x] reg special version type
- [ ] low level codec encode


## Installation 

```bash
go get -u github.com/itering/scale.go
```


## Test

```bash
go test ./... --cover -v
```
